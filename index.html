KitsuGo

Privacy Policy / Política de Privacidad

Last updated: June 13, 2026

1. Introduction

KitsuGo is an anime discovery app that lets you explore popular titles, search for anime, view details, and save your favorites. This policy explains how we handle your information.

2. Data we do NOT collect

KitsuGo does not collect, store, or transmit any personal data to external servers. There are no user accounts, registration, login systems, or tracking mechanisms.

3. Locally stored data

All information generated by the app is stored exclusively on your device and never leaves it:

Data type Storage Purpose
Favorite anime Room Database (local SQLite) Remember your personal favorites list
Search history Room Database (local SQLite) Show recent searches
Dark theme preference DataStore Remember your visual preference
Language preference DataStore Remember your selected language

You can delete this data at any time from system settings (Settings → Apps → KitsuGo → Storage → Clear data).

4. Jikan API usage

KitsuGo consumes the public Jikan API (https://jikan.moe), an unofficial MyAnimeList API. Queries are made from your device directly to Jikan's servers. KitsuGo merely displays the information Jikan returns.

  • We do not send any personal data to Jikan or MyAnimeList.
  • The Jikan API is publicly accessible and free.
  • Review Jikan's Privacy Policy for more details on their operations.

5. App permissions

Permission Usage
INTERNET Query the Jikan API to display anime data

No location, camera, microphone, contacts, external storage, or other permissions are requested.

6. Firebase Analytics (optional)

KitsuGo does not include Firebase Analytics or any other analytics service in its current configuration. If these are added in the future, this policy will be updated and you will be notified via an app update.

7. Third-party services

Service Purpose Privacy policy
Jikan API Fetch anime data jikan.moe/privacy

KitsuGo does not include third-party SDKs (ads, analytics, social networks) that collect data.

8. Security

Since all data remains on your device, there are no risks related to personal data transmission. The app follows recommended Android security practices.

9. Changes to this policy

Any changes to this privacy policy will be reflected by updating this document and, when appropriate, through an in-app notification.

10. Contact

If you have questions about this privacy policy, you can open an issue in the project repository.

Summary: KitsuGo does not collect your data. Everything stays on your device. No accounts, no ads, no tracking.
